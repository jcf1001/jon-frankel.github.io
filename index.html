All Access (HR System)
  • School of Rock is a system of mostly franchise-owned music schools. I worked in the corporate office in Burr Ridge. All Access was an HR system that I built to handle a variety of internal business needs. It exposes certain admin features of Google Apps to school managers -- allowing them to create email accounts, add them to groups, and control access to various other systems. It also allows employees to take training and safety tests, and exposes their progress to their managers. Authentication uses Google Apps accounts, and the system checks for membership in the appropriate school or corporate group to allow access to various functionality. The system also allows corporate to track the progress of compliance across all the schools. Finally, a script can be triggered to audit the system and pass data off to the corporate billing system (Netsuite) to charge schools for the Google Apps accounts licenses that they use.
  • Tech Stack: built in Python for Google App Engine using Ferris, and MVC framework; uses Google's proprietary NoSQL Datastore, with the ndb ORM; connects to Google's APIs via built in libraries for Python on App Engine; Bootstrap for the front end; JQuery for handling requests that use Google's APIs; REST for communication with the billing system, git.
